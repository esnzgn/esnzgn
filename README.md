### 🧠 About Me

- 🎯 Specializing in cancer bioinformatics, synthetic lethality, and multi-omics data
- 🧰 Developer of research tools like **CoPISA**, **NIMAA**, **APSiC**
- 📊 Building visualizations with **ggplot2**, **Shiny**, **igraph**, and **Dash**
